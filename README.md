# ESP8266-AlexaSwitch
Smart switch to control almost anything using Alexa with voice commands. 

## Hardware used
1. ESP-8266 Development Board (NodeMCU)
2. 4-Channel 5V Relay Board
3. Mini Breadboard
4. Dupont wires
5. Amazon Echo Dot

## Softwares
1. Arduino IDE (To upload code to NodeMCU Board)
